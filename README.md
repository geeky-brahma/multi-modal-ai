# Multimodal Reasoning AI Agent

A Streamlit-based application that uses Google's Gemini model to perform multimodal reasoning tasks with uploaded images.

## Demo

https://github.com/user-attachments/assets/600bee1c-0056-4eca-ba76-551020a51486


## Requirements

- Python 3.x
- Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey)

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run multimodal_interface.py
```

1. Enter your Gemini API key in the sidebar
2. Upload one or more files (optional)
3. Provide a task or question
4. Click "Analyze" to get results
